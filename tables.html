Tables

Clean and simple is how I prefer my tables. I always want the data being displayed to stand out and not have the viewer distracted by the styling of the table. There are actually many different styles to choose from in this CSS library when it comes to tables. Each style requires only a simple class at the table level.

NOTE: When building tables, make sure to include <thead> and <tbody> tags, and to use <th> tags for table header cells.

Default Table

The default table style requires no additional markup. Column headers are centered and bolded, and the table header is separated from the content by a thick horizontal line. No other table borders are present.

Markup
<table>
  <thead>
    <tr>
      <th>First Name</th>
      <th>Last Name</th>
      <th>Age</th>
    </tr>
  </thead>
</table>
First Name Last Name Age
John Doe 33
Jane Smith 45
Pete Taylor 18

Vertically Lined Table

Add the .vertical class to the table to get inner vertical borders that extend the height of the table

Markup

...
