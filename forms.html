bpen: skeleton CSS

Forms

This CSS library provides three different types of form layouts that can be activated with a single class at the form level

Default form

On forms with default styling, all form elements are given display:inline-block CSS.

Stacked form

Stacked forms are arranged with the field label directly above the field. Add the .stacked class to the form to activate.

Block form

Block style forms with left aligned labels can be created using the .block-left class on a form and putting each label/input pair in thier own <div> with a .group class

Block style forms with right aligned labels can be created using the .block-right class on a form and putting each label/input pair in thier own <div> with a .group class

Alignment

You can override the alignment of labels by using the .left and .right classes on a label
