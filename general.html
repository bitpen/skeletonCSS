bpen: skeleton CSS

General Styling

Because this is a kind of bare bones CSS library, I've tried to keep the base styling to a minimum. This has allowed me to quickly build on it for each new project without having to figure out any conflicting styles.

Global Layout

Both <html> and <body> have the following margin and padding styling reset to make it easier to style across browsers. 

html,body{
  margin:0;
  padding:0;
}

Fonts

I've generally found that I like to stick to only 2 or 3 fonts per web application. For general heading and paragraph font I use: 

font-family: Georgia, Times, 'Times New Roman', serif;

Headings

For headings, I use a simple stepdown technique of matching font size and line height:

Links

I generally like for in text links to stand out, so they are typically a different color. When hovering on them, they are underlined.

Buttons

<button> , <input type='button'> , and <input type='submit'> all have default styling that makes them stand out as actionable items.

To give a link the same styling, add the .button class to it.

Alignment

Use the classes .left , .right , or .top to align text within any container.
