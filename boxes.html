Boxing

Boxes and blockquotes provide a great way to make important information stand out within the context of the page.

Simple Box

Add the .box class to any <div> to wrap the contents in a borders.

Highlighted Box

Add the .highlight-box class to any <div> to wrap the contents in a borders and give the contents a background color.

Box Title

Give a box a header title.

Markup 
<div class='box>
  <div class='box-title'>
    Markup
  </div>
  Box Content
</div>

Highlighted Box Title

Give a box a highlighted header title.

Markup 
<div class='box>
  <div class='highlight-box-title'>
    Markup
  </div>
  Box Content
</div>

Box Sub-Title

Give a box a sub title.

Markup 
<div class='box>
  <div class='box-title'>
    Markup
  </div>
  Box Content
  <div class='box-subtitle'>
    Box subtitle
  </div>
  Sub Content
</div>

Blockquote

Use .blockquote on a <div> to highlight quoted text or provide a warning or special note of interest.

Code

Use the <code> tag to highlight pieces of code or signify which keys to press.
